# 🕹️ Killer Instinct - Super Famicom (SNES) Emulador Web

¡Bienvenido! Este proyecto es un emulador de Super Nintendo corriendo directamente en el navegador web utilizando **EmulatorJS**. Permite jugar al clásico **Killer Instinct** sin necesidad de instalar programas adicionales.

## 📂 Estructura del Proyecto

El repositorio debe contener la siguiente estructura básica para funcionar:

*   `index.html` - El código de la página web y configuración del emulador.
*   `instingkiller.zip` - El archivo ROM del juego (Super Famicom).
*   `README.md` - Esta guía informativa.

## 🎮 Controles por Defecto

Una vez que cargue el juego, puedes controlar la acción con tu teclado:

| Acción | Tecla |
| :--- | :--- |
| **D-Pad (Moverse/Agacharse)** | Flechas de Dirección (⬆️ ⬇️ ⬅️ ➡️) |
| **Botón A** | Tecla `X` |
| **Botón B** | Tecla `Z` |
| **Botón X** | Tecla `S` |
| **Botón Y** | Tecla `A` |
| **L / R (Gatillos)** | Teclas `Q` / `W` |
| **START** | Tecla `Enter` |
| **SELECT** | Tecla `Shift` |

> 💡 **Nota:** Puedes cambiar los controles o conectar un Gamepad/Control USB haciendo clic en el icono de engranaje (⚙️) dentro del menú inferior del emulador.

## 🛠️ Cómo ejecutar este proyecto en tu computadora

Si quieres probarlo de forma local antes de subirlo a internet:

1. Clona o descarga este repositorio.
2. No abras el archivo `index.html` haciendo doble clic (los navegadores bloquean el emulador por seguridad).
3. Usa un servidor local. Si usas **Visual Studio Code**, instala la extensión **Live Server**, haz clic derecho en `index.html` y selecciona *Open with Live Server*.
